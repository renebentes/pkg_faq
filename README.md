pkg_faq
=============

Pacote de Extensão para gerenciamento de FAQs.

* Componente: com_faq
* Plugin: plg_search_faq
* Módulos: mod_faq_latest, mod_faq_popular

Requirements
------------

* [Joomla! 2.5+](www.joomla.org);
* Template com [Twitter Bootstrap 2.1.0+](www.getbootstrap.com).
